The Anatomy of Mining

Mining is the mechanism by which new blocks are validated and added to a Proof-of-Work blockchain (like Bitcoin). Miners compete using computational power to solve complex cryptographic puzzles. When a miner finds the correct combination (the 'Nonce') that produces a valid 'Hash', they broadcast the block to the network and are rewarded for their work.

🔢

Nonce

A "number used once" that miners continuously change to alter the block's final hash.

#️⃣

Hash

A unique cryptographic digital fingerprint generated (SHA-256) based on the block's contents.

⚒️

Proof of Work

The rule stating a hash must start with specific zeroes (difficulty) to be accepted as valid.

The Power of Immutability

Because every block contains the hash of the previous block, changing a single character in Block 1 invalidates the hash. This instantly breaks the cryptographic link to Block 2, Block 3, and every subsequent block. To alter history, an attacker would have to re-mine the changed block and all blocks after it before the rest of the network produces new blocks—which is mathematically improbable.
